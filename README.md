# JahNoah.com Web Site

This is the official JahNoah website! 

It is an itterative work, to include some info about JahNoah. Also, some way to publish media and other digital content.

Now located on the OnlyJah CICD servers at https://cicd.onlyjah.com/jah/jahnoah-com/, and mirrored to [github](https://github.com/onlyjah/jahnoah-com).

## Initialized With React + TypeScript + Vite

Initialized with `npm create-vite@latest`

- Using Vite's init scaffolding: `Typescript + React Compiler`
- Typechecked Linting: `tseslint.configs.stylisticTypeChecked`

## Added Resources

Static site deploy via gh-pages. See [vite docs](https://vite.dev/guide/static-deploy) fore more details.
